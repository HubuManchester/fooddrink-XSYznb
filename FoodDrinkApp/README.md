# 食光营养助手

食光营养助手是一个基于 .NET MAUI 开发的“食品与饮品”课程项目应用。用户可以记录食品和饮品，查看营养信息，体验移动设备的硬件功能。

## 主要功能

- 食品和饮品列表，支持搜索和详情查看。
- 添加记录表单，包含必填项校验和营养数值验证。
- 调用相机拍摄食品照片并实时预览。
- 使用定位功能记录用餐或购买地点。
- 通过文字转语音朗读营养摘要和帮助内容。
- 使用震动与触觉反馈增强操作提示。
- 支持主题切换（浅色 / 深色）和大字体模式。
- 包含语义标签、屏幕阅读器适配和清晰的验证提示。

## 评分点覆盖

- **UI/UX 与无障碍**：XAML 构建页面，底部导航栏，统一的视觉风格，支持深色模式、语义描述和屏幕阅读器。
- **移动硬件**：相机、定位、文字转语音、震动及触觉反馈。
- **功能完整性**：列表浏览、搜索、添加、详情、设置和硬件演示流程完整。
- **验证与错误处理**：必填项检查、数值合法性校验、权限异常及硬件不可用提示。
- **代码质量**：模型与服务分离，命名规范，服务层可复用，页面职责清晰。
- **部署**：基于 .NET MAUI，支持 Android 和 Windows 双平台。
- **GitHub 使用**：建议持续提交，例如“添加食品列表”、“实现硬件页面”、“增加输入验证”等。

## 运行方式

使用安装有 .NET MAUI 工作负载的 Visual Studio 2022 打开 `FoodDrinkApp.csproj` 或 `FoodDrinkApp.sln`。

推荐演示目标：

- Android 模拟器
- Windows 本机

Windows 构建命令：

```powershell
dotnet build .\FoodDrinkApp.csproj -f net9.0-windows10.0.19041.0
```

Android 构建命令：

```powershell
dotnet build .\FoodDrinkApp.csproj -f net9.0-android
```

本项目通过 `Directory.Build.props` 将构建输出指定到 `C:\MauiBuild\NutriTrack\`，以规避 Android 打包工具在中文路径下出现的 `assets` 路径问题。
